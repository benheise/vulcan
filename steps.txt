Install Ruby dependencies:
yum install gcc-c++ patch readline readline-devel zlib zlib-devel libffi-devel openssl-devel make bzip2 autoconf automake libtool bison sqlite-devel

Download and compile Ruby:
wget https://cache.ruby-lang.org/pub/ruby/2.7/ruby-2.7.1.tar.gz
tar -xvzf ruby-2.7.1.tar.gz; cd ruby-2.7.1/
./configure
make
sudo make install

IMPORTANT: by default ruby install, only your user profile will have Ruby as a path. Ensure your root user also has Ruby in the path or Ruby will not work for the sudo and root account.

Install rubygems:
wget https://rubygems.org/rubygems/rubygems-3.1.4.tgz
tar -xvzf rubygems-3.1.4.tgz; cd rubygems-3.1.4/
sudo ruby setup.rb
(if ruby is not in the sudoers path, might need to su - root and then install)

Install Bundler:
copy bundler-2.1.4.gem to the RHEL box
gem install --force --local *.gem

Once you have bundler installed, cache all gems required for Beef Project by running this in the beef project directory:
On the internet connected system:
bundle package --all --all-platforms
On the real system:
bundle install --local

Once all the Beef Project prerequisites are installed, install beef:
wget https://github.com/beefproject/beef/archive/master.zip
unzip master.zip;cd beef-master/
./install
